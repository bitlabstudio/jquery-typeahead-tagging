jQuery Typehead Tagging - Sample

A jQuery plugin to turn a regular input into a tag input with typeahead autocompletion.

https://github.com/bitmazk/jquery-typeahead-tagging

Code

var languages = ['JavaScript', 'HTML5', 'PHP', 'NodeJS', 'CSS', 'Python', 'Ruby'];

jQuery('#lang').tagging(languages);

Code

var os = ['Mac OS', 'Linux', 'Windows', 'Ubuntu', 'Chrome OS'];

jQuery('#os').tagging(os);
